Given a binary tree, you are supposed to tell if it is a binary search tree. If the answer is yes, try to find the KKK-th smallest key, else try to find the height of the tree.

Format of function:

int CheckBST ( BinTree T, int K );

where BinTree is defined as the following:

typedef struct TNode *BinTree;
struct TNode{
    int Key;
    BinTree Left;
    BinTree Right;
};

The function CheckBST is supposed to return the K-th smallest key if T is a binary search tree; or if not, return the negative height of T (for example, if the height is 555, you must return −5-5−5).

Here the height of a leaf node is defined to be 1. T is not empty and all its keys are positive integers. K is positive and is never more than the total number of nodes in the tree.
