# Advanced Data Structure and Algorithm Analysis [2023-2024 Spring & Summer]

## 课程内容
前半部分主要介绍一些数据结构：如 AVL 树，Splay 树，红黑树，B+ 树，左氏堆，斜堆，二项堆等。

后半部分介绍算法及算法设计和分析的方法，包括摊还分析，回溯，分治，贪心，动态规划，NP 完全理论，近似算法，随机算法，局部搜索，并行算法等。
